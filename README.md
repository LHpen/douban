# douban



##### 简介：

 基于豆瓣TOP250的爬虫和可视化，项目代码对新手比较友好，每段差不多都有注释。



##### 依赖：

python第三方库：

**数据库连接：****pymysql**

**提取数据：** **bs4** **，****re****（正则）**

**发送请求数据：****requests**

**数据清洗：****pandas****，****numpy**

**可视化：****pyecharts****,**

**读取文档：****xlrd**
